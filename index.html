Welcome to MiniWallet

A simple, client-side Ethereum wallet for managing your testnet assets.

Create a new wallet, import an existing one, and send transactions with ease.

🔐 Secure & Private

Your private keys never leave your browser. All operations are performed client-side, giving you full control over your assets.

🚀 Feature Rich

View transaction history, manage a local address book, and see real-time gas fee estimates before you send.

🛠️ Modern Tech

Built with modern JavaScript (ESM) and the lightweight, powerful Viem library for all blockchain interactions.
